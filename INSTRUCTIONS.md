# Usage Instructions

1.  Fork this repo and name it `aftabhussainofficals` (your username — must match exactly).
2.  In your forked repo, go to **Settings → Actions → General**. Under **Workflow permissions**, toggle **Read and Write permissions** and press **Save**. This lets the snake SVG be auto-generated by GitHub Actions.
3.  Go to the **Actions** tab, select **Generate Snake** and click **Run workflow**. This generates your first snake SVG manually; after that a new one is generated every 12 hours automatically.
4.  The **visitor badge** on line 1 is already set to `page_id=aftabhussainofficals.aftabhussainofficals` — no change needed.
5.  Customize the **typing SVG lines** in the `<h1>` block to update your animated headline text.
6.  Customize the **About Me** section (🔭 🌱 💬 ⚡ lines) to match what you're currently building.
7.  Update your **LinkedIn** badge URL if your profile URL changes.
8.  The **Skill Icons** rows use: `python, cpp, c, javascript, html, css, pytorch, tensorflow, git, github, vscode, linux` — edit the `i=` param at `https://skillicons.dev/icons?i=` to add/remove skills.
9.  The **snake SVG** fetches from:
    ```
    https://raw.githubusercontent.com/aftabhussainofficals/aftabhussainofficals/output/github-contribution-grid-snake.svg
    ```
    This path is correct as long as you ran the workflow in step 3 and the `output` branch was created.
10. **Streak stats** use `https://streak-stats.demolab.com/?user=aftabhussainofficals` — this is a public instance and should work without any extra setup.
11. **GitHub Readme Stats** use `https://github-readme-stats.vercel.app/api?username=aftabhussainofficals` — this is the public Vercel instance. For private contributions to count, optionally host your own instance: [instructions here](https://github.com/anuraghazra/github-readme-stats#deploy-on-your-own).
12. The **main.yml** workflow file must be placed at `.github/workflows/main.yml` in your repo for the snake to auto-generate.
13. You're good to go! Leave a ⭐ on the template repo if it helped.
